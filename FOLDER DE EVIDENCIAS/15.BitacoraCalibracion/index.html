BITÁCORA DE CALIBRACIÓN - EVIDENCIAS

Descripción del Proyecto

Sistema CRUD para gestión de calibración de instrumentos con validaciones completas:

  • Validación de emojis y caracteres especiales
  • Validación de scripts maliciosos (XSS)
  • Validación de formato de lecturas (número + unidad)
  • Validación de certificados alfanuméricos
  • Validación de lógica de fechas
  • Validación de longitud de campos
  • Alertas en cliente y servidor

Tecnologías Utilizadas

  • Node.js + Express
  • MySQL2
  • EJS (Embedded JavaScript)
  • Body-Parser
  • dotenv
  • CSS3 (Gradientes, Grid, Flexbox)

Características Principales

  • 7 campos de calibración por instrumento
  • Validaciones en servidor (Node.js)
  • Validaciones en cliente (JavaScript)
  • Diseño responsive
  • Paleta de colores: #6c3483, #39ff14, #ffbe00
  • Mensajes de error mediante alertas
